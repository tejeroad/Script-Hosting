# Script-Hosting
Realización de los scripts necesarios para administrar un hosting propio.

Partimos de que tenemos un hosting montado con servidor DNS, MySQL, Apache y LDAP, también ofrecerá servicio FTP.

Tendremos tres scripts, el de alta (alta.py), el que realizará las bajas (baja.py) y otro que realice una modificación de la contraseña (change_password.py).
